# @arqel/ai

Componentes React do Arqel AI.

Exporta `<AiTextInput>`, par React do PHP `Arqel\Ai\Fields\AiTextField`.
Veja [SKILL.md](./SKILL.md) para o contrato completo de props e
exemplos.

## Instalação

```bash
pnpm add @arqel/ai
```

## Uso

```ts
import '@arqel/ai/register'; // registra no FieldRegistry de @arqel/ui

// ou import direto:
import { AiTextInput } from '@arqel/ai';
```
