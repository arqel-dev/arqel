# @arqel-dev/mcp-server

Official MCP (Model Context Protocol) server for [Arqel](https://arqel.dev) — gives AI assistants access to Arqel docs, project introspection, and scaffolding.

## Status

Iniciativa pós-MVP em desenvolvimento. Roadmap em [`PLANNING/13-pos-mvp-mcp-server.md`](https://github.com/arqel-dev/arqel/blob/main/PLANNING/13-pos-mvp-mcp-server.md).

## Quick start (após release)

```bash
# Claude Code
claude mcp add arqel npx -- -y @arqel-dev/mcp-server

# Cursor / Copilot CLI: ver docs em https://arqel.dev/guide/mcp-server
```
